# Kumbaya Feed

TikTok-style doomscroll feed prototype with spring physics, haptics, comments, and trade drawer.

**Live:** [https://kby-feed.vercel.app](https://kby-feed.vercel.app)

## Run locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) (rewrites `/` → feed).

Cheatsheet: [http://localhost:3000/cheatsheet](http://localhost:3000/cheatsheet)

## Controls

| Key | Action |
|-----|--------|
| `J` / `↓` | Next post |
| `K` / `↑` | Previous post |
| `L` | Like |
| `D` | Dislike |
| `T` | Tip |
| `C` | Comment |
| `S` | Share |

Swipe or scroll on desktop. Hold video for 2× speed.
